# Changelog

All notable changes to this extension are documented here.

This project follows the shape of [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and uses VS Code extension versions from `package.json`.

## Unreleased

### Added

- Documented the local package inspection and dual-registry release workflow for VS Code Marketplace and Open VSX.
- Made the VSIX package boundary explicit so generated `.vsix` archives are not bundled into future extension packages.

## 0.0.1

### Added

- Initial Tables CSV Editor extension for opening `*.csv` files in a VS Code custom editor.
- AG Grid table editing, raw CSV view, GFM Markdown export, and `@khgame/tables` header assistance.
